Руководство пользователя StepListen

Обзор функций

Умное сегментированное обучение

  • 1. Автоматическое разделение аудио на удобные для изучения сегменты
  • 2. Интеллектуальное определение пауз, соответствующее естественным речевым паттернам
  • 3. Гибкий контроль повторений сегментов и скорости воспроизведения
  • 4. Построчная практика повторения для развития навыков аудирования

Точное управление воспроизведением

  • 1. Повторение сложных сегментов одним кликом (Control + R)
  • 2. Клавиша Enter для воспроизведения/паузы, удобно для ведения заметок
  • 3. Клавиши со стрелками для плавной навигации между сегментами
  • 4. Точное позиционирование аудио для освоения каждой детали

Помощник диктовки

  • 1. Построчная запись транскрипции
  • 2. Поддержка импорта/экспорта транскрипции
  • 3. Автоматическое сохранение прогресса
  • 4. Возможность паузы, возобновления и повтора в любое время

Помощник изучения текстов песен

  • 1. Импорт LRC-файлов с автоматической синхронизацией аудио
  • 2. Построчное воспроизведение текста для легкого изучения песен
  • 3. Функция повтора для освоения произношения и интонации
  • 4. Отображение текста для синхронизированной практики пения

Идеально подходит для

  • 1. Любителей иностранных песен
  • 2. Подготовки к языковым тестам (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. Ежедневной практики английского аудирования
  • 4. Изучения подкастов и речей

Стандартные сочетания клавиш

  • ⌘O - Открыть аудиофайл
  • Enter - Воспроизведение/Пауза
  • ↓ - Следующий сегмент
  • ↑ - Предыдущий сегмент
  • ^R - Переключение режима повтора/продвижения
  • ⌘S - Сохранить диктовку
  • F2 - Разделить аудио
  • ^/ - Перейти к первому сегменту

Советы

Если вам не нравятся стандартные сочетания клавиш, вы можете настроить их в Настройки → Клавиатура, нажав на сочетание клавиш и введя новые клавиши

Основные операции

  • 1. Откройте аудиофайл с помощью Command + O или кнопки открытия на панели инструментов
  • 2. Аудио будет автоматически разделено на сегменты. Примечание: Только аудио без фонового шума может быть сегментировано. Слишком много шума повлияет на эффект сегментации.
  • 3. Используйте клавиши со стрелками вверх/вниз для навигации между сегментами
  • 4. Введите диктовку для каждого сегмента
  • 5. Сохраните работу с помощью Command + S

Советы

Режим повтора: Используйте Control + R для включения режима повтора при практике сложных сегментов
Сохранение: При выходе из приложения рекомендуется сохранить содержимое диктовки
Разделение аудио: Нажмите F2 для повторного разделения всего аудиофайла
Функция текстов песен: Вы можете открыть mp3-песню и импортировать текст. Приложение автоматически разделит аудио, чтобы вы могли изучать песню по предложениям
