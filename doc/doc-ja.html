StepListen ヘルプガイド

機能概要

スマートセグメント学習

  • 1. 音声を自動的に学習しやすい長さに分割
  • 2. 自然な会話のパターンに合わせたインテリジェントな休止検出
  • 3. セグメントの繰り返し回数と再生速度の柔軟な制御
  • 4. リスニング力を養うための行ごとの反復練習

精密な再生コントロール

  • 1. 難しいセグメントのワンクリック繰り返し（Control + R）
  • 2. メモを取りやすい再生/一時停止のEnterキー
  • 3. 矢印キーによるスムーズなセグメント間の移動
  • 4. 細部まで習得するための正確な音声位置決め

ディクテーションアシスタント

  • 1. 行ごとの書き起こし記録
  • 2. 書き起こしのインポート/エクスポート対応
  • 3. 進捗の自動保存
  • 4. いつでも一時停止、再開、繰り返し可能

歌詞学習アシスタント

  • 1. LRC歌詞ファイルの自動音声同期インポート
  • 2. 簡単に歌を学べる行ごとの歌詞再生
  • 3. 発音とイントネーションを習得するための繰り返し機能
  • 4. 同期した歌唱練習のための歌詞表示

最適な用途

  • 1. 外国語の歌の愛好家
  • 2. 語学テスト対策（IELTS/TOEFL/Cambridge/CET4-6）
  • 3. 日常的な英語リスニング練習
  • 4. ポッドキャストやスピーチの学習

デフォルトのキーボードショートカット

  • ⌘O - 音声ファイルを開く
  • Enter - 再生/一時停止
  • ↓ - 次のセグメント
  • ↑ - 前のセグメント
  • ^R - リピート/フォワードモード切替
  • ⌘S - ディクテーションを保存
  • F2 - 音声を分割
  • ^/ - 最初のセグメントにジャンプ

ヒント

デフォルトのショートカットキーが気に入らない場合は、設定→キーボードで、ショートカットキーをクリックして新しいキーを押すことで設定変更できます

基本操作

  • 1. Command + O またはツールバーの開くボタンで音声ファイルを開く
  • 2. 音声は自動的にセグメントに分割されます。注意：バックグラウンドノイズのない音声のみ分割可能です。ノイズが多すぎると分割効果に影響します。
  • 3. 上下矢印キーでセグメント間を移動
  • 4. 各セグメントのディクテーションを入力
  • 5. Command + S で作業を保存

ヒント

リピートモード： Control + R で難しいセグメントの練習用リピートモードを切り替え
保存： アプリケーション終了時にディクテーション内容の保存を推奨
音声分割： F2 を押して音声ファイル全体を再分割
歌詞機能： MP3の曲を開いて歌詞をインポートできます。アプリが自動的に音声を分割し、フレーズごとに曲を学習できます
