Funktionsübersicht

Intelligentes segmentiertes Lernen

  • 1. Teilt Audio automatisch in handliche Lernsegmente auf
  • 2. Intelligente Pausenerkennung, die natürlichen Sprachmustern entspricht
  • 3. Flexible Steuerung der Segmentwiederholungen und Wiedergabegeschwindigkeit
  • 4. Zeilenweise Wiederholungspraxis zum Aufbau des Hörverständnisses

Präzise Wiedergabesteuerung

  • 1. Ein-Klick-Wiederholung für schwierige Segmente (Strg + R)
  • 2. Eingabetaste für Wiedergabe/Pause, praktisch für Notizen
  • 3. Pfeiltasten für nahtlose Segmentnavigation
  • 4. Präzise Audiopositionierung zum Meistern jedes Details

Diktat-Assistent

  • 1. Zeilenweise Transkriptionsaufzeichnung
  • 2. Import/Export von Transkriptionen
  • 3. Automatische Fortschrittsspeicherung
  • 4. Jederzeit pausieren, fortsetzen und wiederholen

Liedtext-Lernassistent

  • 1. LRC-Liedtextdatei-Import mit automatischer Audio-Synchronisation
  • 2. Zeilenweise Liedtext-Wiedergabe für einfaches Lernen von Liedern
  • 3. Wiederholungsfunktion zum Meistern von Aussprache und Intonation
  • 4. Liedtextanzeige für synchronisiertes Gesangstraining

Perfekt für

  • 1. Fremdsprachige Musikbegeisterte
  • 2. Sprachtest-Vorbereitung (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. Tägliches Englisch-Hörverständnistraining
  • 4. Podcast- und Redelernen

Standard-Tastenkombinationen

  • ⌘O - Audiodatei öffnen
  • Enter - Wiedergabe/Pause
  • ↓ - Nächstes Segment
  • ↑ - Vorheriges Segment
  • ^R - Wiederholungs-/Vorwärtsmodus umschalten
  • ⌘S - Diktat speichern
  • F2 - Audio teilen
  • ^/ - Zum ersten Segment springen

Tipps

Natürlich können Sie die Standard-Tastenkombinationen anpassen. Gehen Sie zu Einstellungen -> Tastatur, klicken Sie auf eine Tastenkombination und drücken Sie die neuen Tasten

Grundlegende Bedienung

  • 1. Öffnen Sie eine Audiodatei mit Befehl + O oder klicken Sie auf die Öffnen-Schaltfläche in der Symbolleiste
  • 2. Die Audiodatei wird automatisch in Segmente unterteilt. Hinweis: Nur Audio ohne Hintergrundgeräusche kann segmentiert werden. Zu viel Rauschen beeinträchtigt die Segmentierung.
  • 3. Verwenden Sie die Pfeiltasten nach oben/unten zur Navigation zwischen Segmenten
  • 4. Geben Sie Ihr Diktat für jedes Segment ein
  • 5. Speichern Sie Ihre Arbeit mit Befehl + S

Tipps

Wiederholungsmodus: Verwenden Sie Strg + R, um den Wiederholungsmodus für schwierige Segmente zu aktivieren
Speichern: Beim Beenden der App wird empfohlen, den Diktatinhalt zu speichern
Audio teilen: Drücken Sie F2, um die gesamte Audiodatei neu zu teilen
Liedtext-Funktion: Sie können eine MP3-Datei öffnen und den Liedtext importieren. Die App teilt das Audio automatisch auf, sodass Sie das Lied Satz für Satz lernen können
