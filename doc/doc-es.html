Guía de ayuda de StepListen

Descripción general de las funciones

Aprendizaje segmentado inteligente

  • 1. Divide automáticamente el audio en segmentos de aprendizaje manejables
  • 2. Detección inteligente de pausas que coincide con los patrones naturales del habla
  • 3. Control flexible de repeticiones de segmentos y velocidad de reproducción
  • 4. Práctica de repetición línea por línea para desarrollar la intuición auditiva

Control preciso de reproducción

  • 1. Repetición con un clic para segmentos difíciles (Control + R)
  • 2. Tecla Enter para reproducir/pausar, conveniente para tomar notas
  • 3. Teclas de flecha para navegación fluida entre segmentos
  • 4. Posicionamiento preciso del audio para dominar cada detalle

Asistente de dictado

  • 1. Grabación de transcripción línea por línea
  • 2. Soporte para importar/exportar transcripciones
  • 3. Guardado automático del progreso
  • 4. Pausar, reanudar y repetir en cualquier momento

Asistente de estudio de letras

  • 1. Importación de archivos de letras LRC con sincronización automática de audio
  • 2. Reproducción de letras línea por línea para facilitar el aprendizaje de canciones
  • 3. Función de repetición para dominar la pronunciación y entonación
  • 4. Visualización de letras para práctica sincronizada de canto

Perfecto para

  • 1. Entusiastas de canciones extranjeras
  • 2. Preparación para exámenes de idiomas (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. Práctica diaria de comprensión auditiva en inglés
  • 4. Aprendizaje de podcasts y discursos

Atajos de teclado predeterminados

  • ⌘O - Abrir un archivo de audio
  • Enter - Reproducir/Pausar
  • ↓ - Siguiente segmento
  • ↑ - Segmento anterior
  • ^R - Alternar modo de repetición/avance
  • ⌘S - Guardar dictado
  • F2 - Dividir audio
  • ^/ - Saltar al primer segmento

Consejos

Por supuesto, si no te gustan las teclas de acceso rápido predeterminadas, puedes configurarlas en Configuración -> Teclado, haz clic en una tecla de acceso rápido y presiona las nuevas teclas

Operaciones básicas

  • 1. Abre un archivo de audio usando Comando + O o haciendo clic en el botón abrir de la barra de herramientas
  • 2. El audio se dividirá automáticamente en segmentos. Nota: Solo el audio sin ruido de fondo puede ser segmentado. Demasiado ruido afectará el efecto de segmentación.
  • 3. Usa las teclas de flecha arriba/abajo para navegar entre segmentos
  • 4. Escribe tu dictado para cada segmento
  • 5. Guarda tu trabajo usando Comando + S

Consejos

Modo de repetición: Usa Control + R para alternar el modo de repetición para practicar segmentos difíciles
Guardar: Al salir de la aplicación, se recomienda que guardes el contenido del dictado
Dividir audio: Presiona F2 para volver a dividir todo el archivo de audio
Función de letras: Puedes abrir una canción mp3 e importar la letra. La aplicación dividirá automáticamente el audio para que puedas aprender la canción frase por frase.
