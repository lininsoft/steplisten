StepListen 使用指南

功能概览

智能分段学习

  • 1. 自动将音频分割成易于学习的片段
  • 2. 智能检测与自然语言模式相匹配的停顿
  • 3. 灵活控制分段复读次数和播放速度
  • 4. 逐行重复练习培养听力直觉

精确播放控制

  • 1. 一键重复播放困难片段（Control + R）
  • 2. Enter 键播放/暂停，方便记笔记
  • 3. 箭头键在片段间流畅导航
  • 4. 精确的音频定位以掌握每个细节

听写助手

  • 1. 逐行记录听写内容
  • 2. 支持听写内容导入/导出
  • 3. 自动保存进度
  • 4. 随时暂停、继续和重复

歌词学习助手

  • 1. 导入 LRC 歌词文件并自动同步音频
  • 2. 逐行播放歌词便于学习歌曲
  • 3. 重复功能帮助掌握发音和语调
  • 4. 显示歌词进行同步歌唱练习

完美适用于

  • 1. 外语歌曲爱好者
  • 2. 语言考试准备（IELTS/TOEFL/Cambridge/CET4-6）
  • 3. 日常英语听力练习
  • 4. 播客和演讲学习

默认快捷键

  • ⌘O - 打开音频文件
  • Enter - 播放/暂停
  • ↓ - 下一片段
  • ↑ - 上一片段
  • ^R - 切换重复/前进模式
  • ⌘S - 保存听写
  • F2 - 分割音频
  • ^/ - 跳转到第一个片段

提示

如果您不喜欢默认快捷键，可以在 设置→键盘 中点击快捷键并按下新键来设置

基本操作

  • 1. 使用 Command + O 或点击工具栏打开按钮来打开音频文件
  • 2. 音频将自动分割成片段。注意：只有没有背景噪音的音频才能被分割。过多噪音会影响分割效果。
  • 3. 使用上下箭头键在片段间导航
  • 4. 为每个片段输入听写内容
  • 5. 使用 Command + S 保存您的工作

提示

重复模式： 使用 Control + R 切换重复模式来练习困难片段
保存： 退出应用时建议保存听写内容
分割音频： 按 F2 重新分割整个音频文件
歌词功能： 您可以打开 mp3 歌曲并导入歌词。应用将自动分割音频，让您逐句学习歌曲
