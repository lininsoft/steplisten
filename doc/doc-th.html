คู่มือช่วยเหลือ StepListen

ภาพรวมคุณสมบัติ

การเรียนรู้แบบแบ่งส่วนอัจฉริยะ

  • 1. แบ่งเสียงเป็นส่วนย่อยที่เรียนรู้ได้ง่ายโดยอัตโนมัติ
  • 2. ตรวจจับการหยุดอัจฉริยะที่ตรงกับรูปแบบการพูดธรรมชาติ
  • 3. ควบคุมการเล่นซ้ำของส่วนและความเร็วในการเล่นได้อย่างยืดหยุ่น
  • 4. ฝึกซ้ำทีละบรรทัดเพื่อพัฒนาความเข้าใจในการฟัง

การควบคุมการเล่นที่แม่นยำ

  • 1. เล่นซ้ำส่วนที่ยากด้วยคลิกเดียว (Control + R)
  • 2. ปุ่ม Enter สำหรับเล่น/หยุด สะดวกสำหรับการจดบันทึก
  • 3. ปุ่มลูกศรสำหรับการนำทางระหว่างส่วนที่ราบรื่น
  • 4. ตำแหน่งเสียงที่แม่นยำเพื่อเรียนรู้ทุกรายละเอียด

ผู้ช่วยการถอดเสียง

  • 1. บันทึกการถอดความทีละบรรทัด
  • 2. รองรับการนำเข้า/ส่งออกการถอดความ
  • 3. บันทึกความคืบหน้าอัตโนมัติ
  • 4. หยุด เริ่มใหม่ และเล่นซ้ำได้ทุกเมื่อ

ผู้ช่วยการเรียนรู้เนื้อเพลง

  • 1. นำเข้าไฟล์เนื้อเพลง LRC พร้อมซิงค์เสียงอัตโนมัติ
  • 2. เล่นเนื้อเพลงทีละบรรทัดเพื่อการเรียนรู้เพลงที่ง่าย
  • 3. ฟังก์ชันเล่นซ้ำเพื่อฝึกการออกเสียงและทำนองเสียง
  • 4. แสดงเนื้อเพลงสำหรับการฝึกร้องแบบซิงโครไนซ์

เหมาะสำหรับ

  • 1. ผู้ที่ชื่นชอบเพลงต่างประเทศ
  • 2. การเตรียมสอบภาษา (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. การฝึกฟังภาษาอังกฤษประจำวัน
  • 4. การเรียนรู้พอดคาสต์และสุนทรพจน์

ปุ่มลัดมาตรฐาน

  • ⌘O - เปิดไฟล์เสียง
  • Enter - เล่น/หยุด
  • ↓ - ส่วนถัดไป
  • ↑ - ส่วนก่อนหน้า
  • ^R - สลับโหมดเล่นซ้ำ/เดินหน้า
  • ⌘S - บันทึกการถอดความ
  • F2 - แบ่งเสียง
  • ^/ - ไปยังส่วนแรก

เคล็ดลับ

หากคุณไม่ชอบปุ่มลัดมาตรฐาน คุณสามารถตั้งค่าได้ที่ การตั้งค่า → แป้นพิมพ์ คลิกที่ปุ่มลัดและกดปุ่มใหม่

การใช้งานพื้นฐาน

  • 1. เปิดไฟล์เสียงโดยใช้ Command + O หรือคลิกปุ่มเปิดบนแถบเครื่องมือ
  • 2. เสียงจะถูกแบ่งเป็นส่วนโดยอัตโนมัติ หมายเหตุ: เฉพาะเสียงที่ไม่มีเสียงรบกวนพื้นหลังเท่านั้นที่สามารถแบ่งได้ เสียงรบกวนมากเกินไปจะส่งผลต่อการแบ่งส่วน
  • 3. ใช้ปุ่มลูกศรขึ้น/ลงเพื่อนำทางระหว่างส่วน
  • 4. พิมพ์การถอดความสำหรับแต่ละส่วน
  • 5. บันทึกงานของคุณโดยใช้ Command + S
