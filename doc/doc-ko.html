StepListen 도움말 가이드

기능 개요

스마트 세그먼트 학습

  • 1. 오디오를 학습하기 쉬운 크기로 자동 분할
  • 2. 자연스러운 대화 패턴에 맞는 지능형 휴지 감지
  • 3. 구간 반복 횟수와 재생 속도의 유연한 제어
  • 4. 청취력 향상을 위한 줄 단위 반복 연습

정밀한 재생 제어

  • 1. 어려운 세그먼트를 위한 원클릭 반복 (Control + R)
  • 2. 메모하기 편리한 재생/일시정지용 Enter 키
  • 3. 화살표 키로 세그먼트 간 원활한 이동
  • 4. 세부 사항 습득을 위한 정확한 오디오 위치 지정

받아쓰기 도우미

  • 1. 줄 단위 전사 기록
  • 2. 전사 내용 가져오기/내보내기 지원
  • 3. 자동 진행 상태 저장
  • 4. 언제든지 일시정지, 재개, 반복 가능

가사 학습 도우미

  • 1. LRC 가사 파일 자동 오디오 동기화 가져오기
  • 2. 쉬운 노래 학습을 위한 줄 단위 가사 재생
  • 3. 발음과 억양 습득을 위한 반복 기능
  • 4. 동기화된 노래 연습을 위한 가사 표시

최적의 용도

  • 1. 외국어 노래 애호가
  • 2. 어학 시험 준비 (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. 일상적인 영어 듣기 연습
  • 4. 팟캐스트 및 연설 학습

기본 키보드 단축키

  • ⌘O - 오디오 파일 열기
  • Enter - 재생/일시정지
  • ↓ - 다음 세그먼트
  • ↑ - 이전 세그먼트
  • ^R - 반복/전진 모드 전환
  • ⌘S - 받아쓰기 저장
  • F2 - 오디오 분할
  • ^/ - 첫 번째 세그먼트로 이동

팁

기본 단축키가 마음에 들지 않으시면, 설정→키보드에서 단축키를 클릭하고 새로운 키를 눌러 설정할 수 있습니다

기본 조작

  • 1. Command + O 또는 도구 모음의 열기 버튼으로 오디오 파일 열기
  • 2. 오디오가 자동으로 세그먼트로 분할됩니다. 참고: 배경 소음이 없는 오디오만 분할할 수 있습니다. 소음이 너무 많으면 분할 효과에 영향을 미칩니다.
  • 3. 위/아래 화살표 키로 세그먼트 간 이동
  • 4. 각 세그먼트의 받아쓰기 입력
  • 5. Command + S로 작업 저장

팁

반복 모드: Control + R로 어려운 세그먼트 연습을 위한 반복 모드 전환
저장: 앱 종료 시 받아쓰기 내용 저장을 권장합니다
오디오 분할: F2를 눌러 전체 오디오 파일 재분할
가사 기능: MP3 노래를 열고 가사를 가져올 수 있습니다. 앱이 자동으로 오디오를 분할하여 문장별로 노래를 학습할 수 있습니다
