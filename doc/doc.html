StepListen Help Guide

Features Overview

Smart Segmented Learning

  • 1. Automatically splits audio into bite-sized learning segments
  • 2. Intelligent pause detection matching natural speech patterns
  • 3. Flexible control of segment repetitions and playback speed
  • 4. Line-by-line repeat practice to build listening intuition

Precise Playback Control

  • 1. One-click repeat for challenging segments (Control + R)
  • 2. Enter key for play/pause, convenient for note-taking
  • 3. Arrow keys for seamless segment navigation
  • 4. Precise audio positioning for mastering every detail

Dictation Assistant

  • 1. Line-by-line transcription recording
  • 2. Import/Export transcription support
  • 3. Automatic progress saving
  • 4. Pause, resume, and repeat anytime

Lyrics Study Assistant

  • 1. LRC lyrics file import with automatic audio synchronization
  • 2. Line-by-line lyrics playback for easy song learning
  • 3. Repeat function for mastering pronunciation and intonation
  • 4. Lyrics display for synchronized singing practice

Perfect For

  • 1. Foreign song enthusiasts
  • 2. Language test preparation (IELTS/TOEFL/Cambridge/CET4-6)
  • 3. Daily English listening practice
  • 4. Podcast and speech learning

Default Keyboard Shortcuts

  • ⌘O - Open an Audio File
  • Enter - Play/Pause
  • ↓ - Next Segment
  • ↑ - Previous Segment
  • ^R - Toggle Repeat Mode/Forward Mode
  • ⌘S - Save Dictation
  • F2 - Split Audio
  • ^/ - Jump to the first segment

Tips

Of course, if you don't like the default shortcut keys, you can set these shortcut keys, setting->keyboard, click a shortcut key, press the new keyboards

Basic Operations

  • 1. Open an audio file using Command + O or click toolbar open button
  • 2. The audio will be automatically split into segments. Note: Only audio without background noise can be segmented. Too much noise will affect the segmentation effect.
  • 3. Use arrow up/down keys to navigate between segments
  • 4. Type your dictation for each segment
  • 5. Save your work using Command + S

Tips

Repeat Mode: Use Control + R to toggle repeat mode for practicing difficult segments
Save: When exiting the app, it is recommended that you save the dictation content
Split Audio: Press F2 to re-split the entire audio file
Lyrics function: You can open an mp3 song and import the lyrics. The app will automatically split the audio so that you can learn the song sentence by sentence.
